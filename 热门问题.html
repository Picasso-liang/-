算法热门选择题

1. 以下哪种算法最适合求解无向图的最小生成树问题？

A. Dijkstra算法
B. Kruskal算法
C. Bellman-Ford算法
D. 快速排序算法
正确答案：B

解析：Kruskal算法通过排序边并按权重从小到大添加（避圈法），专门用于求解无向图的最小生成树。Dijkstra和Bellman-Ford用于最短路径问题，快速排序是排序算法，与图无关。

2. 二分查找算法的时间复杂度是？

A. O(n)
B. O(n²)
C. O(log n)
D. O(n log n)
正确答案：C

解析：二分查找每次将搜索范围缩小一半，最坏情况下需要log₂n次比较，因此时间复杂度为O(log n)。适用于有序数组的查找。

C++热门选择题

1. 以下关于C++中指针和引用的说法，正确的是？

A. 指针可以为空，引用不能为空
B. 指针可以被重新赋值，引用也可以
C. 指针和引用都需要手动释放内存
D. 指针和引用的大小相同
正确答案：A

解析：引用必须在定义时初始化且不能指向空值，而指针可以为空；引用一旦初始化就不能改变指向的对象，指针可以重新赋值；引用本身不分配内存，无需手动释放；指针大小通常是4或8字节，引用的大小取决于所指向对象的类型。

2. C++中，下列哪个关键字用于定义常量？

A. static
B. const
C. final
D. volatile
正确答案：B

解析：const用于定义常量，被修饰的变量值不能被修改；static用于修饰静态变量或函数；final用于限制继承或重写（C++11后）；volatile用于提示编译器变量可能被意外修改，避免优化。

Python热门选择题

1. Python中，以下哪个不是可变数据类型？

A. list
B. dict
C. str
D. set
正确答案：C

解析：Python中str（字符串）是不可变类型，一旦创建不能修改其内部元素；list（列表）、dict（字典）、set（集合）是可变类型，可直接修改内部元素。

2. 以下哪个语句可以正确导入math模块并使用其sqrt函数？

A. import math.sqrt
B. from math import sqrt
C. include math
D. use math.sqrt
正确答案：B

解析：Python中导入模块的语法有两种：①import 模块名（使用时需加模块名前缀，如math.sqrt()）；②from 模块名 import 函数名（可直接使用函数名，如sqrt()）。A语法错误，C、D不是Python的导入关键字。

Java热门选择题

1. Java中，以下关于接口和抽象类的说法，错误的是？

A. 接口中只能有抽象方法，抽象类中可以有非抽象方法
B. 一个类可以实现多个接口，但只能继承一个抽象类
C. 接口和抽象类都不能被实例化
D. 接口中的方法默认是private的
正确答案：D

解析：Java接口中的方法默认是public abstract的（Java 8后可有默认方法），不能是private；抽象类可以有private方法。其余选项均为正确表述。

2. Java中，以下哪个关键字用于抛出异常？

A. try
B. catch
C. throw
D. finally
正确答案：C

解析：throw用于手动抛出异常对象（如throw new Exception()）；try用于包裹可能抛出异常的代码；catch用于捕获并处理异常；finally用于定义无论是否发生异常都执行的代码块。
